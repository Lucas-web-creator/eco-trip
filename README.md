# eco-trip
Sistema de análise de pegada de carbono em transportes. Calcula emissões de CO2 para diversos modelos de veículos, ajudando usuários a fazer escolhas mais sustentáveis
.Este projeto foi desenvolvido como parte do Bootcamp Formação GitHub Copilot da DIO PRO, representando o módulo final (Desafio de Projeto) do programa. O objetivo é disponibilizar uma calculadora interativa que quantifica o impacto ambiental gerado por diferentes tipos de viagens, permitindo aos usuários tomar decisões mais conscientes e sustentáveis.
